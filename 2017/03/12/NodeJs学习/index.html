title:node学习—前提基础知识

什么是JavasScript?

  • 脚本语言；运行在浏览器中；一般用来做客户端页面的交互

    JavaScript的运行环境

  • 运行在浏览器内核中的JS引擎

    浏览器中的JavaScript可以做什么？

  • 操作DOM（对DOM的增删改注册事件）
  • AJAX/跨域
  • BOM（页面跳转，历史记录，console.log(),alert()）
  • ECMAScript

    浏览器中的JavaScript不可以做什么？

  • 文件操作（文件和文件夹的CRUD）
  • 没有办法操作系统信息
  • 由于运行环境特殊（我们写的代码是在不认识的人的浏览器中运行）

    在开发人员能力相同的情况下编程语言的能力取决于什么？

  • 语言本身
  • 语言本身只是提供定义变量，定义函数，定义类型，流程控制，循环结构之 类的操作
  • 取决于运行该语言的平台（环境）
  • 对于JS来说，我们常说的JS实际上是ES，大部分能力都是有浏览器的执行引擎决定
  • BOM和DOM可以说是浏览器开放出来的接口
  • 比如：Cordova中提供JS调用摄像头，操作本地文件的API
  • Java既是语言也是平台
  • Java运行Java虚拟机（跨操作系统）
  • PHP既是语言也是平台
  • C#语言 平台： .net Framework
  • C#可以运行在MONO这样的平台
  • 因为有人需要将C#运行在Linux平台，所以出现了MONO

NodeJS

NodeJs是一个基于ChromeV8引擎的Javascript运行环境。NodeJS使用了一个事件驱动，非阻塞式I/O的模型，轻量又高效。NodeJS的包管理工具npm，是全球最大的开源库生态系统。

Chrome V8最大的特点就是单线程

总结

  • nodejs是一个Javascript的运行环境（平台），不是一门语言，也不是一个JS框架。
  • node的实现结构
  • node可以用来开发服务端应用程序，web系统
  • 基于node的前端工具集
